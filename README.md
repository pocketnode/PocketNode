# PocketNode
[![](https://img.shields.io/badge/Chat-On%20Discord-738BD7.svg?style=for-the-badge)](https://discord.gg/VT928ah)

## What's This?
PocketNode is a server software for Minecraft: Bedrock written in node.js

## Running
First Run:
```
$ npm i && node start
```

Afterwards:
```
$ node start
```
